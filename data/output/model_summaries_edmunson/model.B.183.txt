tools for analyzing icechipseq datasets this repository contains scripts and tools for analyzing icechipseq data as described by grzybowski et al. 2015 and shah et al. 2018. dependencies and required files these scripts assume use of a unix system and require the following tools to be installed and added to the path environment variable bowtie2 bedtools samtools ucsctools we have tested these scripts on ubuntu lts 12.04 14.04 16.04 and 18.04. for the above listed tools we have tested bowtie2 v. 2.3.4.1 bedtools v. 2.26.0 and samtools v. 1.7. the scripts folder contains the icechip and computehmdanderror scripts for analyzing the icechipseq data.
