bash run as root cd mycontainer runc create mycontainerid view the container is created and in the created state runc list start the process inside the container runc start mycontainerid after 5 seconds view that the container has exited and is now in the stopped state runc list now delete the container runc delete mycontainerid this allows higher level systems to augment the containers creation logic with setup of various settings after the container is created andor before it is deleted.
