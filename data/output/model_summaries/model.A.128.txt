follow these steps for easy installation open instal new software dialog help → install new software... add a new update site fill in any name eg imageexport fill httpveger.github.comeclipsegefimageexport in the location field select the newly created location select at least eclipse gef imageexport plugin and eventually other plugin to get additional image formats finish the installation by clicking next a couple of times and accepting the license using the plugin to export a gef diagram make sure that a gef based editor is active open file → export → general → imageexport choose a filetype from the available image types note depending on the installed imageexport plugins more type might be available choose a location to save the image click finish the exported image will exactly fit the contents of the editor scaled at 100. if larger images are required it is adviced to export as svg using the imageexport.svg plugin and scale in another program to the desired scale.
